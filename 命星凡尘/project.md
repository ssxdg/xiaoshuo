# 命星凡尘

## 作品设定

### 世界观

- 时间：修仙世界，无明确年代
- 地点：青州（林家所在地）、青云山脉（青云仙宗）、中天修仙界（后续展开）
- 修炼体系：经典九境（练气→筑基→金丹→元婴→化神→炼虚→合道→大乘→渡劫）
- 灵根体系：传统灵根（单/双/三/四灵根）→破命灵根（命星玉融合）

### 核心体系

**命星玉**
- 上古至宝，可融合灵根、感知命格、参悟天机
- 能够吸收他人气运，转化为自身修为
- 与青云仙宗有千丝万缕的联系
- 本质是一件“道器”，与修仙者道心相连，随道心成长而进化

**命格与气运**
- 命格：天地规则的一部分，每个人的命格生来注定，无法更改
- 命星玉可打破命格束缚
- 凡人信仰之力可突破渡劫瓶颈，打破“修仙者不问凡俗”的设定

**道心铸器**
- 修士可将自身道心、执念融入法器，打造专属“道器”
- 林砚尘的本命法器：砚心剑
- 以命星玉为引，以自身道心为魂
- 随境界与道心成长自动进阶：破命格枷锁、碎天地规则

**心魔设定**
- 心魔非纯粹恶：自身执念、前世因果、天地规则的倒影
- 可斩、可渡、可同化
- 修心道路线

### 主角

- 姓名：林砚尘
- 年龄：16岁（出场时）
- 出身：林家旁支子弟
- 灵根：四灵根（被命星玉融合为破命灵根，表现为七彩灵根）
- 性格：沉稳内敛、不卑不亢、坚定果决
- 金手指：命星玉
- 本命法器：砚心剑
- 修炼功法：破道诀（从命星玉传承获得）
- 道：“破”——破命格、破枷锁、破天地规则
- 目标：打破命运枷锁，掌握自己的命运
