- 标题方向：黑潮探路 / 雾下回声 / 旧址前哨。

## 七、写作规则备忘

1. 正文真实非空白字符不少于3000。
2. 每章先蓝图，至少4个完整场景，2个外部事件，1个内部变化。
3. 每章必须写明因果链、主角代价、下一章钩子。
4. 每章只推进2-3条核心伏笔。
5. 不要用“加油、必胜、永不言弃”式口号凑收束；用具体行动、代价和后果替代。
6. 完章后运行：

```powershell
powershell -ExecutionPolicy Bypass -File "skills文件/ai-novel-creation/scripts/validate_chapter.ps1" -Path "命星凡尘/第N卷_xxx/第N章_xxx.md"
```
