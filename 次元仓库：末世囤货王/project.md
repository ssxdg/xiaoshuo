# 次元仓库：末世囤货王

## 作品设定

### 世界观
- 时间：2030年1月
- 地点：中国某二线城市
- 末世类型：寒潮气温骤降 + 丧尸病毒爆发 + 停水断电秩序沦陷

### 系统设定
- 次元仓库系统
- 基础功能：无限囤货、保鲜恒温、分类储物
- 升级解锁：活体养殖、土地种植、简易加工
- 获取方式：末世前三天意外觉醒

### 人物
- 主角：陈默（28岁，普通上班族）
- 性格：谨慎、内向、善于观察
- 特点：单身、有囤积癖、喜欢看末日生存小说

### 章节规划
- 1-8章：末世前三天（系统觉醒、囤货准备）
- 9-25章：末世初期（苟发育、收小弟）
- 26-40章：势力建立（城堡、小镇）
- 41-50章：最终决战与收束
